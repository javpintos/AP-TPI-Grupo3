<h2>Ejecutar de forma local</h2>

<h3>Clonar el projecto</h3>

```
git clone https://github.com/javpintos/AP-TPI-Grupo3
```
<h3>Instalar las dependencias del pom.xml</h3>

```
mvn clean install
```
<h3>Ejecutar el servidor con maven</h3>

```
mvn spring-boot:run
```
